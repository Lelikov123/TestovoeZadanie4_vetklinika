# Ветеринарная клиника
Это решение тестового задания 4 
Консольное приложение для управления ветеринарной клиникой на Java.

## Функционал

### Для владельцев животных:
- Регистрация и вход
- ### Добавление/удаление животных
- ### Запись к ветеринару
- Просмотр истории посещений

### Для ветеринаров:
- Просмотр приемов
- ### Работа с медкартами
- Добавление записей о лечении
- История болезней животных

## Запуск
1. Установите SQL Server Management Studio
2. Создайте базу данных VeterinaryClinic
3. Настройте подключение в Database.java
4. Запустите Main.java
